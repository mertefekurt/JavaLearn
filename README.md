JavaLearn is a collection of standalone Java modules for practicing the language from core syntax to functional programming. Each file is runnable on its own, making it easy to read a concept, compile it, change it, and observe the result.

![Code snapshot](assets/code-snapshot.png)

## Modules

| File | Topic |
| --- | --- |
| `Module01_BasicSyntax.java` | Primitive types, operators, strings, casting, and numbers |
| `Module02_ControlStructures.java` | Branching and loops |
| `Module03_MethodsAndClasses.java` | Methods, classes, and object basics |
| `Module04_ArraysAndCollections.java` | Arrays and collection APIs |
| `Module05_InheritanceAndPolymorphism.java` | Inheritance and overriding |
| `Module06_ExceptionHandling.java` | Exceptions and defensive flows |
| `Module07_InterfacesAndAbstractClasses.java` | Interfaces and abstract classes |
| `Module08_GenericsAndTypeSafety.java` | Generics and typed containers |
| `Module09_FileIOAndStreams.java` | File I/O and stream handling |
| `Module10_MultithreadingAndConcurrency.java` | Threads and concurrency primitives |
| `Module11_LambdaAndFunctionalProgramming.java` | Lambdas, functional interfaces, and streams |

## Quick Start

```bash
git clone https://github.com/mertefekurt/JavaLearn.git
cd JavaLearn
javac Module01_BasicSyntax.java
java Module01_BasicSyntax
```

Run another module by replacing the file and class name:

```bash
javac Module11_LambdaAndFunctionalProgramming.java
java Module11_LambdaAndFunctionalProgramming
```

## Study Flow

1. Read the module from top to bottom.
2. Predict the console output.
3. Compile and run the file.
4. Change one section and rerun it.
5. Move to the next module when the concept is clear.
